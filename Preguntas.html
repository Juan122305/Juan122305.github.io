What is HTML?

Es un lenguaje de programacion que define la estructura de las paginas web. Es decir se usa para el codigo de diseño web.

Head and body tags

Estos son parte de la estructura de un documento HTML, el head contiene informacion sobre la pagina, mientras que el body contiene todo el contenido de toda la pagina.

Explain the section, article, aside, header, footer tag

Son elementos semanticos que se utilizan para estructurar el contenido de una pagina web.

Section: Representa una seccion generica de un documento, se utiliza para diferenciar secciones y tambien sirve para determinar que contenido corresponde a que parte de un esquema.

Article: Se utiliza para agrupar contenido que se pueda reutilizar en otras partes de la web o aplicacion.

Aside: Representa una seccion de una pagina que contiene informacion que esta relacionada con el contenido principal pero que se puede separar de este contenido.

Header: Determina la cabecera de una web o de un elemento, se utiliza para identificar contenido que introduce un bloque de codigo HTML y se puede usar en cada seccion que lo necesite.

Footer: Define el pie de pagina y tambien se utiliza para indicar informacion menos importante sobre la seccion, como contenido relacionado o datos de la web.

What is CSS?

Es un lenguaje de programacion que controla el aspecto de las paginas web. Se encarga de definir como se presentan los elementos de un sitio web, como los colores, las fuentes, los margenes y la disposicion de los objetos.

What is the purpose of CSS?

Es separar el contenido de la presentacion de un sitio web, es decir se encarga de definir todo el estilo visual de la pagina web.

How do you apply a CSS to an HTML?

Esto se puede aplicar en linea, es decir se aplica directamente en los elementos HTML usando el atributo style. Otra opcion es en CSS interno se escribe dentro de una etiqueta style en la seccion de head del documento HTML. Por ultimo es en CSS externo, es decir se coloca en un archivo .css separado se enlaza con el HTML mediante la etiqueta link, esta es la mejor practica para mantener un codigo limpio y reutilizable.

What is a selector? What is a CSS rule?

Un selector es el mecanismo que se usa para identificar los elementos HTML a los que se aplicara un conjunto de estilos, este permite seleccionar uno o varios elementos en la pagina para darles formato.

Una regla CSS es un bloque de codigo que define como se vera un elemento HTML, una regla esta compuesta por un selector y un bloque de declaraciones.

Explain a Tag Selector, Class Selector, Tag and Class Selector, Id Selector

Tag Selector: Selecciona todos los elementos de un tipo especifico en el documento HTML.

Class Selector: Selecciona elementos con una clase especifica, este se define con un punto seguido del nombre de la clase.

Tag and Class Selector: Este selector aplica estilos solo a los elementos de un tipo especifico que tengan una clase en particular.

Id Selector: Selecciona un unico elemento con un ID especifico, se usa con el simbolo # seguido del nombre ID.

What is a class? What is an id? How are they related to CSS?

Una clase es un atributo HTML que permite asignar los mismos estilos a multiples elementos mediante CSS, se define con el atributo class en HTML y se selecciona en CSS usando un punto. Un ID es un identificador unico en HTML que se usa para dar estilos o manipular un solo elemento en la pagina, se define con el atributo id y en CSS se selecciona usando #.

Explain the CSS Box Model?

Describe como se estructuran y se comportan los elementos en la pagina web, este es fundamental para entender el diseño web y la distribucion de los elementos, asi controlando los margenes, bordes, relleno y contenido.

How can you change the font color using CSS?

Para esto se tiene que usar la propiedad color, se puede aplicar de diferentes maneras: usando un selector de etiqueta, usando una clase, usando un ID, usando inline CSS y por ultimo usando valores de color en diferentes formatos.
